# Summary

「ことばの探索機」は『ワードバスケット』というカードゲームで使えることばを見つけるWebアプリです。画面から手札の文字を入力すると、その文字を先頭か末尾に入れたことばを教えてくれます。

http://kototan.info/

# Requirements

- Ruby 2.2.1.
- Rails 4.2.1
- Firebase
